# RecommendationSystem

RecommendationSystem é o sistema desenvolvido para o trabalho de conclusão de curso 
sobre o tema "Análise e Aplicação de Inteligência Artificial no Desenvolvimento
de um Algoritmo de Recomendaçõa de Conteúdo" para o curso de engenharia elétrica
com ênfase em computação da escola politécnica da USP.

## Clonando o repositório

Para realizar a clonagem desse projeto em sua máquina local, rode o seguinte comando:

```bash
https://github.com/Nyito/RecommendationSystem.git
```

## Instalando depdendências

Antes de rodar o projeto, verifique se possui as seguintes dependências :

Python 3.12.3
Pip

Para instalar os pacotes necessários para o projeto, rode o seguinte comando:

(É opcional a criação de um ambiente virtual)
```bash
python -m venv venv
venv\Scripts\activate
```

```bash
pip install -r requirements.txt
```

## Utilização

Para rodar o sistema de recomendação, navegue até o diretório main:

```bash
cd main 
```

Dentro do diretório main, rode o seguinte comando:
```bash
streamlit run app.py
```

O sistema estará disponível para utilização em seu navegador no endereço "localhost:8501"
